# 🗡️ QuestPack

QuestPack es un plugin ficticio para demostrar sistemas de misiones en servidores Minecraft.

---

## 🚀 Características
- Misiones configurables con recompensas  
- Soporte para misiones diarias y semanales  
- Integración con economía y rangos  
- Configuración en `config.yml`  

---

## 📦 Instalación
1. Coloca `QuestPack.jar` en la carpeta `plugins/`.  
2. Reinicia el servidor.  
3. Configura misiones y recompensas en `config.yml`.  

---

## 📜 Comandos
| Comando   | Descripción             | Permiso            |
|-----------|-------------------------|--------------------|
| `/quest`  | Administra misiones     | `questpack.quest`  |
